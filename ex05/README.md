# 第5回
## 負けるな！こうかとん(ex05/fight_koukaton.py)
### 使い方＆説明
* 左右の矢印キーでこうかとんを動かす。
* 敵（教授）の赤本をよけながらGPAを稼ぐ。
* スペースを押すことでペンを射出出来、赤本を相殺する事が出来る。
### 操作方法
* スペースキー：ペン（銃弾）を発射
* 左矢印キー:こうかとんを左へ動かす
* 右矢印キー:こうかとんを右へ動かす
### 出来たこと
* 1:背景、銃弾、敵の攻撃の変更。
* 2:こうかとんがペンで戦う事が出来る。
* 3:一度に打ち込めるペンの数を変更。
* 4:敵を倒すごとにGPAが上がる。
