## サザエさんクイズ(ex01/quiz.py)
### 遊び方
* 解答をして正解なら「正解！」、不正解なら「不正解！」表示される。
* 最低一問は表示される。
### プログラム無いの解説
* main関数：クイズプログラムの全体の流れを担当する。
* shutudai関数：ランダムに選んだ問題を出題し、解答をmain関数に返す。
* kaitou関数：解答と正解をチェックし、結果を出力する。
